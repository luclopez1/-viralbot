   - Pexels: https://www.pexels.com/api/
   - Detalles: Lee API_SETUP.md

PASO 3: Configura .env
   - Edita: C:\Users\luclo\youtube-ia-automation\.env
   - Rellena: ANTHROPIC_API_KEY, YOUTUBE_API_KEY, PEXELS_API_KEY

PASO 4: Prueba sin API keys (opcional)
   python test_demo.py

PASO 5: Genera tu primer video real
   python generate_content.py

PASO 6: Automatiza (opcional)
   npm install -g n8n
   n8n start
