## APIs que Necesitas (GRATIS)

### Claude (Anthropic)
- **Obtén API key**: https://console.anthropic.com
- **Docs**: https://docs.anthropic.com
- **Límite Free**: Créditos iniciales (suficiente para empezar)

### YouTube Data API
- **Console**: https://console.cloud.google.com
- **Documentación**: https://developers.google.com/youtube/v3
- **Límite Free**: 10,000 unidades/día (suficiente para uploads)

### Pexels (Imágenes)
- **Obtén API key**: https://www.pexels.com/api/
- **Docs**: https://www.pexels.com/api/documentation/
- **Límite Free**: 200 requests/hora (ilimitado en total)
